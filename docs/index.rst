pygnusocial: GNU Social API for Python 3
========================================

pygnusocial uses `Requests <https://github.com/kennethreitz/requests/>`_
and `Requests-OAuthlib <https://github.com/requests/requests-oauthlib>`_
libraries to provide an easy-to-use Python interface for building clients for
GNU Social.

Installation
------------

pygnusocial can be installed with `pip <https://pip.pypa.io/>`_:

::

  $  pip install gnusocial
